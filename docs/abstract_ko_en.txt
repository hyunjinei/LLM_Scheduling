Korean Abstract

본 연구는 Job Shop Scheduling Problem(JSSP)에 대한 대규모 언어모델(LLM) 기반 스케줄링에서, 문제 전체의 스케줄을 한 번에 문자열로 생성하는 기존 end-to-end 접근이 갖는 실행 가능성 위반, 출력 편향, 그리고 학습 범위를 벗어난 문제 크기에 대한 일반화 한계를 다룬다. 기존 LLM 기반 조합최적화 연구는 자연어로 기술된 문제를 직접 해결할 수 있는 가능성을 보여주었으나, 많은 접근이 여전히 whole-schedule autoregressive generation과 함께 sampling, reflection, repair, 또는 Best-of-N과 같은 사후 보완 절차에 의존하여 실행 가능성을 개선하는 구조에 머물러 있다. 이에 비해 본 연구는 JSSP를 전체 해 생성 문제가 아닌 step-by-step action selection 문제로 재정의하고, 각 의사결정 단계에서 현재 상태와 실행 가능한 행동 집합을 바탕으로 다음 행동 하나를 선택하는 LLM 프레임워크를 제안한다. 이를 위해 환경이 계산한 feasible action set 위에서만 디코딩이 이루어지도록 하는 feasible-action masking을 도입하여, 모델링된 행동 공간 내부에서 실행 불가능한 행동이 생성 단계에서 직접 차단되도록 설계한다. 또한 절대 job index에 대한 출력 편향을 완화하기 위해 randomized step-local action code를 도입하고, 선택된 행동을 환경에 즉시 적용하여 상태를 갱신함으로써 모델의 출력이 실제 스케줄 진행 과정과 직접 대응되도록 한다. 학습 단계에서는 teacher schedule로부터 각 의사결정 시점의 상태-행동 쌍을 추출하여 step-wise supervision dataset을 구성하고, 이를 기반으로 LoRA 기반 policy fine-tuning을 수행한다. 나아가 완성된 스케줄을 사후적으로 분석하여 비효율적 선택이 발생한 의사결정 단계를 다시 검토하고 수정하는 self-reflection prompting을 적용할 수 있으며, reinforcement learning을 통해 makespan 중심의 정책 개선으로 확장할 수 있도록 설계한다. 본 연구는 행동 단계에서 실행 가능성을 직접 통제하면서도 makespan 최소화를 지향하는 step-by-step LLM 기반 JSSP 프레임워크를 제시한다.


English Abstract

This study addresses key limitations of large language model (LLM)-based scheduling for the Job Shop Scheduling Problem (JSSP), particularly the feasibility violations, output bias, and limited out-of-scale generalization that arise when existing end-to-end approaches generate an entire schedule as a single text sequence. Recent LLM-based combinatorial optimization studies have demonstrated the potential of solving natural-language-described problems directly, but many of them still rely on whole-schedule autoregressive generation together with post-hoc mechanisms such as sampling, reflection, repair, or Best-of-N selection to improve feasibility. In contrast, this work reformulates JSSP not as a full-schedule generation problem but as a step-by-step action selection problem, and proposes an LLM framework that selects exactly one next action at each decision step based on the current state and the feasible action set. To this end, we introduce feasible-action masking, which restricts decoding to the feasible action set computed by the environment and directly blocks infeasible action generation within the modeled action space. In addition, we introduce randomized step-local action codes to reduce output bias tied to absolute job indices, and immediately apply each selected action to the environment so that the model output directly corresponds to the actual scheduling progression through state updates. During training, we construct a step-wise supervision dataset by extracting state-action pairs from teacher schedules and perform LoRA-based policy fine-tuning on top of this representation. Furthermore, the framework is designed to support self-reflection prompting, in which a completed schedule is analyzed after rollout so that inefficient decisions can be revisited and revised, and it can also be extended with reinforcement learning for makespan-oriented policy improvement. Overall, this work presents a step-by-step LLM framework for JSSP that directly controls feasibility at the action level while pursuing makespan minimization.
